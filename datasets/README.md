# Datasets

This folder contains datasets used across multiple machine learning notebooks in this repository.

## World University Rankings Dataset

**File:** World University Rankings.csv  

**Used in Notebooks:**
- Visualisation  
- Linear Regression  
- K-Means Clustering  
- KNN Classifier  
- Decision Tree Classifier  
- Dimensionality Reduction (PCA)  
- Random Forest  

**Description:**  
This dataset contains global university ranking information across multiple performance indicators, including teaching quality, research output, citations, international outlook, and industry income.  

**Applications:**  
- Exploratory Data Analysis (EDA)  
- Regression modeling  
- Classification modeling  
- Unsupervised learning (clustering & PCA)  
- Feature engineering and performance evaluation  

**Source:**  
Public academic dataset (global university ranking data).

## Body Performance Dataset

**File:** bodyPerformance.csv  

**Used in Notebooks:**
- Non-linear Regression  
- Decision Tree Regressor  

**Description:**  
This dataset contains physical performance and physiological attributes used to model and predict body performance metrics.  

**Applications:**  
- Regression modeling  
- Feature engineering  
- Model evaluation and performance comparison  

**Source:**  
Public fitness and performance dataset.

## Clustered Comments Dataset

**File:** clustered_comments.csv  

**Used in Notebooks:**
- Transformer  

**Description:**  
This dataset contains clustered textual comments used for training and evaluating Transformer-based deep learning models for sequence modeling and text classification tasks.

**Applications:**  
- Natural Language Processing (NLP)  
- Sequence modeling  
- Transformer-based deep learning  

**Source:**  
Public text dataset.
